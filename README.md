# PasswordManager
| STILL IN PROGRESS|
|------------------|

This project made in c++ using the sqlite3.c librairy can be used to store your passwords effectively. I have secured it with a system of a database that store a password that you've decided and if the password the user enter is the same you can access to the database where you store your passwords. I still need to make more clear for the user and to really clean the code. But for a first time working with databases in c++ I'm quite proud of it. I also need to add a lot of securities to it so that any user can use it without any prior experience of SQL at all.

If you want to use it, first of all  thank you, you'll need to download each files that are not inside the include folder. Do not worry about any .db files because it will automaticaly create it for you. You can either run the exe with the terminal with this command ./passwordManager theNameYouWantYourUserDatabaseToBe.db or by simply double click on the file, if you use the second version you will be asked for the name of the file where your user password will be stored, you need to pass something that end with ".db" for it to be a database file.
